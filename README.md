# 💨 Tailwind Project – Desktop Website Design

This is a **website design project** built using **Tailwind CSS**.  
The project focuses on creating a clean and modern UI using a utility-first approach.

⚠️ Currently, this project is designed for **desktop view only**. Responsive design is not yet implemented.

## 🌐 Live Preview
👉 https://md-sakil352.github.io/tailwind-project/

## 📌 Project Overview
- Complete website layout using Tailwind CSS  
- Clean and modern UI design  
- Desktop-first approach  
- Structured and readable code  

## 🚀 Features
- Modern UI design  
- Built with Tailwind CSS utility classes  
- Well-organized layout  
- Easy to customize  

## 🛠️ Technologies Used
- HTML5  
- Tailwind CSS  

## 🎯 Purpose
This project was created to practice **Tailwind CSS** and understand how to build a full website layout using a utility-first framework.

## ⚠️ Limitations
- Not responsive yet  
- Optimized for desktop view only
